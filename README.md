# Interoperabilit-tsprojekt

### Structure

1. ``vs-code-plugin/`` : The VS Code extension
2. ``CLIs/`` : The CLIs that will be called by the VSCode extension
3. ``Beckhoff/`` : Beckhoff Parsers (XML -> C# and C# -> XML)
4. ``TIA-parser/`` : TIA Portal Parser (.db -> C#)
5. ``TagGenerator/`` : Legacy TIA Portal Parser (AML -> C# and C# -> AML)
6. ``Testing-Files/`` : A folder with sample files for testing
